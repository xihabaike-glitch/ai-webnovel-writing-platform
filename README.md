# AI Webnovel Writing Platform

AI 网文写作平台项目。

当前阶段以产品与技术文档为主，目标是构建一个面向网文作者的 AI 写作生产平台：传统写作工作台负责作品、章节、大纲、人物、世界观和伏笔管理，AI 模型负责审稿、补强、改写和平台适配。
