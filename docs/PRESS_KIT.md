# Press Kit

## GitHub Release 文案

AI Webnovel Writing Platform 正式开源。

这是一个面向网文创作者的开源 AI 写作生产平台：让 AI 像编辑部一样协作，让作者像主编一样掌控网文生产。

项目锁定八个平台：起点中文网、番茄小说、七猫、晋江文学城、知乎盐选、WebNovel、Royal Road、Wattpad。它不是纯聊天机器人，也不是一键生成百万字小说，而是把大树结构、平台打法、AI 编辑部、任务回执、失败修复和投稿包放进同一条可验收流水线。

AI 输出始终是草稿、候选稿、审稿意见、任务回执或复盘材料，最终采用权始终在作者手里。

## 7 天 AI 网文首章样本挑战

1. 选一个目标平台。
2. 建一个作品项目。
3. 写出钩子、结尾承诺和故事树。
4. 跑一次首章样本。
5. 审稿并二改。
6. 记录作者接受或拒绝。
7. 发布复盘。

这不是自动投稿挑战，而是验证 AI 能否在作者主导下稳定支持真实创作。
